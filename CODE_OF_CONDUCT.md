# Stablecoin Stack Foundation - Code of Conduct

All participants in the **Stablecoin Stack Foundation** community are expected to **treat each other with respect and professionalism**. This applies to all repositories, discussions, issues, pull requests, tutorials, and any public communication channels.

---

## 1. Be Respectful

- Respect different opinions, approaches, and experiences.
- Focus on ideas, not individuals.
- Avoid personal attacks, insults, or offensive language.
- Feedback should be constructive and actionable.

---

## 2. Inclusive Communication

- Use **clear, inclusive, and accessible language**.
- Avoid jokes or examples that could be offensive based on gender, race, religion, sexual orientation, disability, or other personal traits.
- Respect cultural differences.

---

## 3. Collaboration & Professionalism

- Be patient with and suportive to newcomers and contributors at all skill levels.
- Provide clear instructions when reviewing or suggesting changes.
- Follow the **contribution and governance guidelines**.
- Work in good faith to resolve conflicts.

---

## 4. Handling Violations

If you encounter behavior that violates this Code of Conduct:

1. Report it to the **maintainers** or **foundation leadership** via private message or email.
2. Include relevant context (links, screenshots, or descriptions).
3. Maintain confidentiality; do not escalate publicly without guidance.

Violations may result in:

- Warnings
- Temporary restrictions
- Removal from project contributions
- Escalation to foundation leadership

---

## 5. Scope

This Code of Conduct applies to:

- All repositories under the **Stablecoin Stack Foundation**
- GitHub issues, pull requests, and discussions
- Tutorials, documentation, and guides
- Public communications, meetings, or presentations representing the foundation

---

By participating in the **Stablecoin Stack Foundation**, you agree to follow this Code of Conduct.
